# Stock Analyzer Ex

#### Goal:
Implementation to analyze stock
* 抓取上市櫃每日盤後資料以及財報(月、年)作 基本面和技術面分析
* 抓取即時報價做高頻交易

### How do I get set up? ###

ex:
*Please input begin day: ex:2017/7/30
*2017/6/1
*Please input end day:
*2017/7/31

*如此便會下載"上市櫃該區間的盤後資料"並將之存入到資料庫[未完成]
